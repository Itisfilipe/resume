# Filipe Amaral - Resume

**View:** [PDF](resume.pdf) | [Markdown](resume.md)

## Export

```bash
npm install
npm run export:pdf    # Generate PDF and Markdown
```

## Structure

- `resume.json` - Resume data
- `resume.pdf` - Generated PDF version
- `resume.md` - Generated Markdown version
- `render-resume.mjs` - Custom export script
